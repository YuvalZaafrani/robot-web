# קריטריוני קבלה (QA Checklist) - רובוט.פרו

מסמך זה מרכז את הבדיקות הנדרשות כדי לוודא שכל דרישות המטלה והשדרוגים בוצעו בהצלחה.

## 1. דרישות המטלה (Must-Have)

### ON-PAGE SEO
- [ ] **H1 בדף הבית**: מכיל בדיוק את הביטוי "רובוט שואב שוטף מומלץ לשימוש בדירה קטנה" כמחרוזת רציפה.
- [ ] **Metadata**: לכל עמוד (`/`, `/about`, `/guide`, `/compare`, `/faq`, `/article`, `/sources`) יש `title` ו-`description` ייחודיים בעברית.
- [ ] **Canonical**: לכל עמוד מוגדר תג canonical המצביע ל-URL הנכון תחת `https://robot-web-six.vercel.app`.
- [ ] **Internal Linking**: קיימים קישורים פנימיים בין המדריך, ההשוואה, השאלות הנפוצות והמאמר.
- [ ] **עברית מלאה**: אין טקסטים באנגלית ב-UI (כולל הודעות שגיאה של וידאו).

### Technical SEO
- [ ] **Sitemap**: הקובץ `sitemap.xml` זמין ומכיל את כל 7 הנתיבים המרכזיים.
- [ ] **Robots.txt**: הקובץ `robots.txt` זמין ומאפשר אינדוקס של כל האתר ומצביע לסייטמאפ.
- [ ] **JSON-LD Article**: מופיע בדף `/article` ומכיל נתונים תקינים.
- [ ] **JSON-LD FAQ**: מופיע בדף `/faq` ומכיל את כל השאלות והתשובות.
- [ ] **OpenGraph/Twitter**: מוגדרים לכל עמוד (כולל תגית `images`).

### OFF-PAGE
- [ ] **מסמך תכנית**: הקובץ `docs/offpage-backlinks-plan.md` קיים ומכיל 10 יעדים עם טיוטות בעברית.

### Affiliate (AliExpress)
- [ ] **Rel Attributes**: כל קישורי האפילייט כוללים `rel="nofollow sponsored noopener"`.
- [ ] **Target**: כל קישורי האפילייט נפתחים בחלון חדש (`target="_blank"`).
- [ ] **Disclosure**: מופיע גילוי נאות בעברית ליד כפתורי ה-CTA ובדפים הרלוונטיים.

## 2. שדרוגי UX ועיצוב (הערות המשתמש)

### דף הבית (Landing Page)
- [ ] **וידאו**: מוגדל (aspect-square בדסקטופ), רץ בלופ ללא overlay "עיגול".
- [ ] **תוכן**: הדף כולל סקציות של "למה דירה קטנה", "קריטריונים", "טעויות נפוצות", ו-"FAQ teaser".
- [ ] **השוואה**: קיים אזור השוואה בדף הבית שניתן לקפוץ אליו דרך anchor (`#compare`).

### עמוד השוואה (/compare)
- [ ] **Visual Cards**: תצוגת כרטיסים הכוללת תמונה, ציון כללי, ופירוט תכונות.
- [ ] **Progress Bars**: מופיעים לכל דגם ומציגים ניקוד לפי גובה, שקט, ניווט ושטיפה.
- [ ] **טבלה טכנית**: מופיעה מתחת לכרטיסים להשוואה יבשה של נתונים.

### מדריך ומאמר (/guide, /article)
- [ ] **פירוט**: המדריך והמאמר כוללים שלבי בחירה, דוגמאות מדידה וצ'קליסטים.
- [ ] **TOC**: המאמר כולל תוכן עניינים עם קישורי קפיצה (anchors) לסעיפים.
- [ ] **עיצוב**: שימוש ב-Design System אחיד (כרטיסים, אייקונים, צבעי Accent).

### מקורות (/sources)
- [ ] **אמינות**: מופיעים קישורים ליצרנים (Roborock, Dreame) ומעבדות (RTINGS, Vacuum Wars).
- [ ] **הבחנה**: קישורי מקורות רגילים הם ללא `sponsored`, וקישורי חלקי חילוף הם עם `sponsored`.
